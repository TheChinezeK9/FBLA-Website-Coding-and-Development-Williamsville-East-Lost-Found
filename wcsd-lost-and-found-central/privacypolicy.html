Privacy Policy

This policy explains how East High School Lost & Found handles the limited information needed to help students recover missing items.

1. Data Minimization
This student project project strictly follows the principle of data minimization. The project only collects the minimum amount of personal data that is necessary, like the school email, to carry out the lost and found project.

2. Credential Security
Our project's login mechanism focuses on the security of the users. There is no practice of data harvesting or "scraping." The user credentials provided during the login process are only used to manage the user sessions and are never used to access the user credentials for any other purpose.

3. Zero Third-Party Sharing
Our project follows the principle of zero sharing, which is also aligned with the education mission of the project. There is no sharing of user data, contact information, or product descriptions with third-party advertisers or anyone else.

4. User Sovereignty
In this project, the user is the supreme authority, and the user data is never stored beyond the time required to handle the lost item or the term of the school. The user has the right to delete the account and data at any time, which will be honored immediately.
